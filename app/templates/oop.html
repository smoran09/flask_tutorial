Object Oriented Programming

Intro

Object oriented programming, or OOP, is a programming model which uses classes and objects. It implements inheritance polymorphisms,abstract, and encapsulation entities. The main purpose of OOPs is to bind data and the functions to work as one so that no other part of code can access the data.

Abstract

  • Abstract allows to create a set of methods that are created within any child classes built from the abstract class. In calculator, I used @staticmethod, which differs from @abstractmethod since @staticmethod can be written before classes, instance variables, and instance methods while @abstractmethod can only be written in front of classes and methods. The image shows an example of @abstractmethod.

Encapsulation

Encapsulation wraps data and the methods that work on data within one unit. The image details encapsulation used in the calculator. Encapsulation restricts accessing variables and methods which prevents accidental editing of data.
