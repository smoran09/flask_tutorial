AAA Testing

Arrange-Act-Assert

Aids in writing a successful and strong code. Helps create tests to do so.

[Image. From Samantha Moran Nov.21,2021)]

Arrange

  • Arrange sets up the test case. It inputs and targets prior to the start of the test.

[Image. From Samantha Moran Nov.21,2021)]

Act

  • Act covers the main of what is to be tested. Examples include a function or method, calling a REST API, or interacting with a web page. It focuses on the target behavior.

[Image. From Samantha Moran Nov.21,2021)]

Assert

  • Assert steps verify the quality of the response as good or bad. Assertions may check numeric or string values or otherwise, they may require checking multiple facets of a system. Assertions determine if the test passes or fails.

[Image. From Pixabay by designwebjae. Jan 26 ,2018. (https://pixabay.com/images/id-3109378/)]

Conclusion

AAA testing is the process of a code. It is an important process which ensures success of the code. In my example above, the calculator was set to complete division. Through AAA testing, the coding was successful.
